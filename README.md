# Random Color Generator (JavaScript Project) <br>
<br>
This is a beginner-friendly JavaScript project that generates random RGB colors on button click. <br>
<br>
When the user clicks the button: <br>
- The background color of the body changes randomly <br>
- The button color also changes randomly <br>
- The RGB color value is displayed on the screen <br>
- The RGB value is automatically copied to the clipboard <br>
<br>
## Features <br>
<br>
Random RGB color generation using JavaScript <br>
Dynamic DOM manipulation <br>
Clipboard copy functionality using Clipboard API <br>
Clean and simple UI <br>
Beginner-friendly code structure <br>
<br>
## Technologies Used <br>
<br>
HTML <br>
CSS <br>
JavaScript (DOM + Clipboard API) <br>
<br>
## How It Works <br>
<br>
A function generates random RGB values using Math.random() <br>
The color is applied to the document body and button using style.backgroundColor <br>
The RGB value is displayed inside a paragraph element <br>
The same RGB value is copied to the user's clipboard using navigator.clipboard.writeText() <br>
